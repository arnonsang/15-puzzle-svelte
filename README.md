

# Svelte 15 Puzzle Game!
Live demo [here](https://15puzzle.games.iamickdev.com)

## To run this project
- Install package using `npm i` or `pnpm i`
- Run development `npm run dev` or `pnpm run dev`
- It's work with [Bun](https://bun.sh/) too!

## Stack

- SvelteKIT
- TailwindCSS (+daisyui)
